# Field Primers

> Minimum-background notes per Millennium problem. The goal of each
> file is to give a non-specialist enough vocabulary and conceptual
> scaffolding to *read this repository's attempts on that problem*,
> not to give a substitute for graduate study.

Each file follows the same skeleton:

1. **Problem statement** — the formal CMI statement, paraphrased
   with care.
2. **Why it is hard** — the headline obstructions.
3. **Minimum vocabulary** — terms the reader must recognize.
4. **Prior results to know about** — the few that come up
   repeatedly in attempts.
5. **Failed approaches and barriers** — the dead ends and the
   meta-theorems that close them off.
6. **Recommended starting reading** — at most ~5 references.
7. **Glossary deltas** — entries in
   [`../glossary.md`](../glossary.md) and
   [`../notation.md`](../notation.md) that originate from this
   field.
