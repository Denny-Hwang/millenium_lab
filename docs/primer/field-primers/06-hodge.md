## 1. Problem statement

TODO: On a smooth projective complex algebraic variety $X$, every
Hodge class is a rational linear combination of cohomology classes
of algebraic cycles. Paraphrase carefully — there are several
variants ($\mathbb{Q}$-Hodge, integral Hodge, ...); only the
rational version is the official CMI statement, and the integral
Hodge conjecture is known to be false.

## 2. Why it is hard

TODO: The conjecture identifies a topological / Hodge-theoretic
class (defined via harmonic forms) with an algebro-geometric class
(defined via subvarieties). The two definitions live in different
worlds; bridging them requires either constructing subvarieties
from cohomology classes (hard) or understanding which Hodge
classes can possibly be algebraic (also hard).

## 4. Prior results to know about

TODO:

- Lefschetz (1924): the $(1,1)$ case — every Hodge class in
  $H^2$ is algebraic. This is the foundational known case.
- Cattani–Deligne–Kaplan (1995): algebraicity of the locus of
  Hodge classes; a deep conditional result.
- Voisin (2002): a counterexample to the integral version.
- Many known cases for special varieties (abelian varieties of
  small dimension, complete intersections, ...).
