# 06 — Hodge Conjecture

CMI 공식 진술 요지: **매끄러운 사영 복소 다양체 위에서, 모든 Hodge class는
대수적 cycle의 유리수 선형결합이다.**

본 디렉토리에서는 다음 방향을 병렬 추적합니다.

- 표준 추측(Grothendieck) 및 동기 이론과의 관계
- abelian variety, K3 표면 등 부분 사례
- ℓ-adic / p-adic 비교
