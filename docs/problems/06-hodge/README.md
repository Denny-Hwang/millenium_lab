# 06 — Hodge Conjecture

CMI official statement (gist): **On a smooth projective complex variety,
every Hodge class is a rational linear combination of algebraic
cycles.**

This directory tracks the following directions in parallel:

- Standard conjectures (Grothendieck) and their relationship to motive
  theory
- Partial cases such as abelian varieties and K3 surfaces
- ℓ-adic / p-adic comparisons
