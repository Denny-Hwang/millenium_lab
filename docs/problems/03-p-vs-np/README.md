# 03 — P vs NP

CMI 공식 진술 요지: **다항 시간 검증 가능한 모든 결정 문제가 다항 시간 풀이를
가지는가?** (즉, P = NP인가?)

대다수의 합의는 P ≠ NP이지만, 본 프로젝트는 양 방향 모두에 대해 시도를
허용하되, **현재 알려진 모든 장벽**(relativization, natural proofs,
algebrization)을 통과하는지를 첫 검증으로 봅니다.
