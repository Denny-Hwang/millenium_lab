# 03 — P vs NP

CMI official statement (gist): **Does every decision problem verifiable
in polynomial time also have a polynomial-time algorithm?** (i.e., does
P = NP?)

The community's consensus is P ≠ NP, but this project allows attempts in
either direction. The first verification gate is whether an attempt
passes through **all currently known barriers** (relativization, natural
proofs, algebrization).
