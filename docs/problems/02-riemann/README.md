# 02 — Riemann Hypothesis

CMI official statement (gist): **All non-trivial zeros of the Riemann
ζ-function have real part 1/2.**

This directory is the entry point for how this project approaches the
problem.

## Entry Guide

1. [`statement.md`](statement.md) — official statement
2. [`equivalent-forms.md`](equivalent-forms.md) — equivalent forms (prime number theorem error term, Mertens function, etc.)
3. [`known-results.md`](known-results.md) — partial results (zero density, zeros on the critical line, functional equation, etc.)
4. [`failed-approaches.md`](failed-approaches.md) — past failed attempts and their reasons
5. [`verification-bar.md`](verification-bar.md) — verification criteria specific to this problem
6. [`status.md`](status.md) — progress history

## Candidate Consilient Fields

- Analytic number theory (classical)
- Random matrix theory (GUE statistics)
- Noncommutative geometry (Connes program)
- Dynamical systems (Hilbert–Pólya conjecture variants)
- Formal languages / combinatorics

Bridges are tracked under `bridges/`.
