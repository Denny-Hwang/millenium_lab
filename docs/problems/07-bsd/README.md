# 07 — Birch and Swinnerton-Dyer Conjecture

CMI official statement (gist): **For an elliptic curve E, the order of
vanishing of L(E, s) at s = 1 equals the rank of the Mordell–Weil group
E(ℚ), and the precise leading coefficient is given by the BSD
formula.**

This directory tracks the following in parallel:

- Weak BSD (rank ↔ order of vanishing)
- Strong BSD (the precise constant)
- Connections to modular forms and Iwasawa theory
