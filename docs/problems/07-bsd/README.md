# 07 — Birch and Swinnerton-Dyer Conjecture

CMI 공식 진술 요지: **타원곡선 E의 L-함수 L(E, s)의 s=1에서의 차수가
Mordell–Weil 군 E(ℚ)의 랭크와 일치하며, 정확한 leading coefficient는 BSD
공식에 의해 주어진다.**

본 디렉토리에서는 다음을 병렬 추적합니다.

- 약한 BSD (랭크와 차수의 일치)
- 강한 BSD (정확한 상수)
- modular form, Iwasawa 이론과의 연결
