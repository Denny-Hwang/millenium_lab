# 01 — Poincaré Conjecture

This problem is **already solved**. Grigori Perelman's 2002–2003 ArXiv
preprints (Ricci flow with surgery) supplied the essential proof, which
was later verified; in 2010 CMI awarded him the prize (declined).

This directory is **reference-only**. New attempts cannot be merged.

---

## Role of This Directory

- Document how Ricci flow and surgery techniques inspire the other six
  problems in this project.
- Serve as a **reference example** of how an "already solved problem" is
  represented in the repository pipeline.
- If byproduct bridges are discovered (e.g., Poincaré ↔ general-dimension
  topology), move them to `bridges/`.
