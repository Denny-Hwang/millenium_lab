# 01 — Poincaré Conjecture

본 문제는 **이미 해결된 문제**입니다. Grigori Perelman이 2002~2003년에 ArXiv에
공개한 일련의 논문(Ricci flow with surgery)이 본질적 증명을 제공했고, 이후
검증을 거쳐 CMI는 2010년 Perelman에게 상을 수여했습니다(본인은 사양).

본 디렉토리는 **참고용**으로만 운영됩니다. 새 attempt는 머지되지 않습니다.

---

## 본 디렉토리의 역할

- Ricci flow와 surgery 기법이 어떻게 본 프로젝트의 다른 6개 문제에 영감을
  주는지 정리.
- "이미 해결된 문제"가 본 저장소 파이프라인에서 어떻게 표현되는지의 **레퍼런스
  사례**.
- 부산 다리(예: Poincaré ↔ 일반 차원 위상수학) 후보가 발견되면 `bridges/`로
  이전.
