# 05 — Navier–Stokes Existence and Smoothness

CMI official statement (gist): **For the 3D incompressible Navier–Stokes
equations, do smooth solutions exist for all time, or do they blow up in
finite time?**

This problem is central to PDE analysis. The repository tracks the
following directions in parallel:

- Partial regularity theory (successors of Caffarelli–Kohn–Nirenberg)
- Concrete construction attempts of blow-up solutions
- Surrogate models (Burgers, dyadic models)
