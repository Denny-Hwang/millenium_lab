# analysis/ — 종단적 메타 분석

본 디렉토리는 본 저장소 활동의 **종단적 메타 분석**을 보관합니다. 어떤 영역이
효과적이었는지, 어떤 다리가 실제 진척을 가져왔는지를 데이터 기반으로 본 디렉토리
에 집계합니다.

> ⚠️ 본 디렉토리는 **자동 갱신 대상**입니다. 직접 편집하지 말고
> `scripts/build-data.py`, R5(주간 보고), R6(cross-attempt synthesis)를 통해
> 갱신해 주세요. 직접 편집은 R6의 분석 결과를 덮어쓸 수 있습니다.
