# analysis/ — Longitudinal Meta Analysis

This directory holds the **longitudinal meta analysis** of repository
activity. It aggregates, in a data-driven way, which areas were
effective and which bridges actually produced progress.

> ⚠️ This directory is **auto-updated**. Do not edit it directly;
> refresh it via `scripts/build-data.py`, R5 (weekly report), or R6
> (cross-attempt synthesis). Direct edits can be overwritten by R6's
> output.

## Outputs (placeholder)

- [`per-problem-progress.md`](per-problem-progress.md) — per-problem progress time series.
- [`bridge-effectiveness.md`](bridge-effectiveness.md) — which bridges led to candidates.
- [`meta-observations.md`](meta-observations.md) — meta observations (outcome distribution, model efficiency, etc.).

`weekly/<YYYY-Www>.md`,
`cross-synthesis/<problem>-<date>.md`, and similar reports accumulate
over time.
